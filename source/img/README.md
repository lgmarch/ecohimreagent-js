Папка для изображений.

Все SVG-иконки предназначенные для спрайта находятся в `img/icons/`:
```
img/
  icons/
    vk.svg
```

Остальная векторная и растровая графика содержится в папке `img/`:
```
img/
  bg.jpg
  hero.png
  burger.svg
```

Графику можно группировать папками в `img/`:
```
img/
  catalog/
    product-1.jpg
    product-2.jpg
  form/
    check.jpg
    uncheck.jpg
  bg.jpg
  hero.png
  burger.svg
```
